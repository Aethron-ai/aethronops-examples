## Why AVM?

Azure Verified Modules are:
- Officially maintained by Microsoft
- Security-reviewed and compliance-tested
- Versioned and backward-compatible
- Used by 10,000+ organizations worldwide

AethronOps does NOT generate raw Terraform resources. Every Azure service
is deployed through its official AVM module, ensuring:
- Consistent security defaults
- Automatic best-practice configuration
- Traceable module versions (see table above)
